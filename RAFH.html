OVERVIEW

Room Away From Home is a mobile application that enables user to book a room designated for COVID-19 isolation. The app features a listing page, self assessment tool, and a daily self checkup to aid in Quarantine Anxiety. Room Away From Home was one of my project User Interface Design class in SFU.
